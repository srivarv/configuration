# Sri's Configuration Scripts

This repository contains a collection of my configuration scripts used to initialize new installs of Linux (specifically Ubuntu). 

# Quickstart

```bash
$ git clone ...
$ cd config
$ chmod +x configure.sh
$ ./configure.sh
```
